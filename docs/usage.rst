Basic Usage
===========

Starting ShellP
---------------
To start ShellP, use this command::

   python3 -m shellp

If ``~/.local/bin`` is in your ``$PATH``, you can also use this command::

   shellp

Exiting ShellP
--------------
To exit ShellP, type the ``exit`` command or send an EOF (``Ctrl+D`` on Linux).

..versionadded:: 0.2.0
   The ability to exit using an EOF
